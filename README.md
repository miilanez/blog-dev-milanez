# Blog Dev Milanez

Este projeto é um aplicação Web no formato de blog, construído por mim para aprendizado e desenvolvimento no Front-End. O projeto foi construído utilizando as seguintes ferramentas: 

- NextJs
- JavaScript
- Tailwind CSS
- GraphQL (HyGraph)
- React Icons
